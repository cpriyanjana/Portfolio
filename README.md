# 2. Vehicle Speed Analysis – Rock Island, IL

### Overview  
An interactive **Shiny dashboard** analyzing real-world vehicle speed data collected manually to evaluate speeding behavior and traffic safety concerns.

### Key Features
- Speed category classification (Within Limit / Slightly Over / Significantly Over)
- Student-level observation comparison
- Car-type based speed analysis
- Scatter plots and summary statistics
- Interactive filtering

### Major Findings
- Over **67% of vehicles exceeded the speed limit**
- SUVs and Sedans were the most frequent violators
- Significant variation between observers

### Tools & Skills
- R Shiny  
- ggplot2  
- dplyr  
- Reactive programming  
- Data summarization  
- Dashboard development  

### Impact
Demonstrated:
- Data-driven safety evaluation
- Behavioral pattern detection
- Interactive public-facing dashboard design
